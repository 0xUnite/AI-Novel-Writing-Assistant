# 第三批：核心配角卡片定稿

## 周阿禾

条目名：周阿禾角色卡

表面身份：周衍病弱妹妹，药谷底层棚屋里靠药吊命的人。

真实身份（受保护字段）：药谷本地弱灵枢承压者；她不是天外血脉，也不是转世者，而是天痕压损灵枢在普通人身上的提前显影。她的身体等于盖天裂缝压迫底层人的第一张账单。

情感轨迹：前 1-10 章是周衍必须活下去的近身牵挂；11-30 章被药钱、试药童和周家税刀不断推成周衍的道德底线；中后期她不承担战力挂件功能，而承担“周衍不能把所有人命都算成账”的回拉功能。

语言风格：句子短，气息弱，但不装无知；她常先看出周衍在撒谎或硬撑，再用很轻的一句话拦他。她不讲大道理，多说“别去”“我知道”“哥，你又没睡”这类低声、贴身、让人心口一紧的话。

推演依据：第 4 章“灵枢受损……得用借命草吊着” + 第 4 章“胸口起伏得几乎看不见的呼吸” + 第 9 章“矿谷深处忽然裂开一道极短的天痕” + 数据库角色表“靠药吊命 / 活到哥哥不必再为一株草拿命去赌的时候”

推演逻辑：阿禾病因已指向灵枢受损 + 天痕在近处落下 + 她的目标不是变强而是不拖累 → 她的真实身份应是天痕承压者，不是血统奇迹。

决策备注：枪毙“星族遗孤/神女转世/天命炉鼎”，均犯血统或工具人套路。

## 宁见微

条目名：宁见微角色卡

表面身份：黑市里看人极毒、从不白给的生意人。

真实身份（受保护字段）：黑市账路枢纽的低位掌柜；她掌握税契、暗路、验货口和旧井货流，却不是黑市真正的主人。她的价值在于知道“谁敢收、谁不敢收、哪条路只能走一次”，并敢把自己的一部分未来押在会涨价的人身上。

真实立场：利润优先的灰盟友。她不会无条件救周衍，但当周衍证明自己能把死人账、玄铁、坠星残片和季寒庐的破绽串成可兑现的局，她会押注，并要求未来回报。

语言风格（50-100 字）：她说话像估价，先压价，再留缝；嘴上慢，眼睛快。她很少把威胁说死，常用“你可以不卖”“我不收，是现在不收”逼对方自己补出真实底牌。

推演依据：第 5 章“黑市里有” + 第 5 章“我收了，转手就得被上面的人盯上” + 大纲第 14 章“宁见微不肯白给路” + 大纲第 20 章“只能走一次的暗路路引”

推演逻辑：她懂玄铁风险 + 掌握暗路路引 + 要周衍未来办事 → 她是账路枢纽型灰盟友，不是好心掌柜。

决策备注：枪毙“黑市女王/宗门卧底/无偿贵人”，问题是过早抬高或削弱交易感。

## 裴照庭

条目名：裴照庭角色卡

表面身份：前期只以名字、玄铁价值和闻人星阑线投影到盖天法界的人。

真实身份（受保护字段）：浑天星海裴系掌权者，正在用闻人星阑、祖地坐标和星相仪补全更高层的星族名分合法性。他不是地方矿主，也不是盖天法界本地裴家。

真实目的：拿到祖地、星相仪和闻人星阑三样钥匙。玄铁只是低阶饵料和识人筹码，能筛出谁知道货、谁急着脱手、谁背后可能连着星相仪残片。

接触周衍的动机：裴照庭愿意让玄铁流到药童手里，不是善意交易，而是低成本投石。一个底层药童若只为阿禾药钱卖掉玄铁，就只是普通耗材；若能看出玄铁不能贱卖，并沿玄铁摸到废炉区、星相仪和闻人星阑，才值得被裴系继续盯住。

语言风格：雍容、冷、像在签一份对方看不懂的契。裴照庭不急着骂人，也不直接炫耀力量；他会把最残酷的交换说成最体面的秩序，让对方在答应之后才意识到自己已经失去退路。

推演依据：第 3 章“裴照庭说过，这东西能换几十两银子” + 第 3 章“裴照庭自己都要靠闻人星阑换回星族名分” + 数据库角色表“拿到祖地、星相仪和闻人星阑三样钥匙” + 大纲第 231 章“追查裴照庭为何对闻人星阑穷追不舍”

推演逻辑：玄铁被裴照庭定价 + 他需要闻人换名分 + 后续追查其追杀闻人的原因 + 数据库明确三钥匙目标 → 玄铁交易是诱饵和筛选，不是帮药童。

决策备注：枪毙“本地裴家少主/单纯玄铁商/无脑反派”，均与星海三钥匙线矛盾。

## 闻人星阑

条目名：闻人星阑核心卡补充

表面身份：第 9 章自天痕坠落、重伤被困下界的清冷女子。

真实身份（受保护字段）：浑天星海没落远古星族传人，掌握通往内核祖地的旧坐标，并在后续被确认持有能改变星海秩序的祖地钥匙。

情感轨迹：前 10 章与周衍是互相利用的脆弱同盟；31-40 章身份外露，知识交换开始有代价；121-130 章进入星族余脉线；231-240 章祖地钥匙线被裴照庭逼到台前。

语言风格：清冷、少字、先观察再回答；她不会把高阶知识倾倒给周衍，而是只给足以让他做选择的信息。关键时刻比周衍更敢切断退路。

推演依据：设定文档第 81-84 行“高维跌落型大女主……没落远古星族的传人” + 大纲第 9 章“自天外坠落” + 大纲第 37 章“确认闻人星阑来自浑天星海的没落远古星族” + 大纲第 231 章“闻人星阑掌握的是能改变星海秩序的祖地钥匙”

推演逻辑：天外坠落 + 星族传人 + 祖地坐标与祖地钥匙 → 闻人星阑是星相仪、祖地和裴照庭三线的核心。

决策备注：枪毙“师姐型金手指说明书”，因会让她依附主角；她必须保留独立目标。

## 主角身边核心配角最终名单

条目名：核心配角三人组

定稿内容：主角身边核心配角封顶三人：周阿禾、闻人星阑、宁见微。

选择理由：周阿禾负责底层人命与情感底线；闻人星阑负责天盖之外的认知跃迁与星族主线；宁见微负责黑市、账路和灰色交易方法。三人分别对应“不能只算账的人”“账外还有天的人”“账必须兑现的人”。裴照庭不列入核心配角，因为他是中后期主反派和外部压迫源。

推演依据：数据库角色表“周阿禾：核心配角” + 设定文档第 81-84 行“单一核心女主角” + 数据库角色表“宁见微：灰盟友” + 数据库角色表“裴照庭：第二三卷主反派”

推演逻辑：情感底线 + 认知跃迁 + 灰色交易 + 反派排除 → 核心配角只能是阿禾、闻人、宁见微三人。

决策备注：枪毙“裴照庭入队”，因反派职责明确；枪毙多人团，因会稀释底层压迫。

## 第三批自审报告

1. 本批所有条目是否都有推演链：是。所有真实身份、立场、语言风格都有前 10 章、设定文档、大纲或数据库角色表依据；没有把无依据身份写成事实。

2. 套路黑名单检查：无“神族后裔/天命之子/转世”；闻人星阑是既有“没落远古星族传人”，不是本批新增血统爽点；阿禾明确不是星族血脉。

3. 五个锚点回收：本批回收阿禾病弱真实原因、裴照庭为什么愿意和药童交易玄铁；周衍胸口和星相仪地窖由第二批回收；矿沟死人由第四批伏笔清单回收。

4. 候选内部枪毙样例：枪毙“阿禾是星族遗孤”，犯血统论且抢闻人星阑功能；枪毙“宁见微是黑市女王”，前 10 章只支持低位掌柜和账路枢纽；枪毙“裴照庭是本地裴家少主”，与星海名分和三钥匙目标冲突。

5. 与前批内容矛盾检查：与第一、第二批一致。阿禾的病因承接“天痕压损灵枢”；裴照庭一系承接第一批“星海暗线”；闻人星阑承接设定文档与大纲祖地线。
