# 第二批：修炼体系细节

## 三阶段境界划分

条目名：灵枢段

定稿内容：灵枢段对应盖天法界的凡阶三境：拓脉境、铭骨境、筑宫境。拓脉境打通阻塞灵枢，只能靠药材、灵石和低等灵气续命；铭骨境开始把劣质符文刻进凡骨，但这只是“假命骨”，还没有真正进入浑天星海的命骨体系；筑宫境在体内筑起灵力核心，触到叹息天盖，是下位界域承载极限。

推演依据：设定文档第 30-33 行“灵枢是修士体内与外界灵气进行物质交换的核心器官群” + 设定文档第 49-52 行“拓脉境 / 铭骨境 / 筑宫境” + 大纲第 100 章“真正打穿天盖并活着出去”

推演逻辑：灵枢负责底层交换 + 凡阶三境均在盖天法界 + 筑宫触碰天盖 → 灵枢段完整覆盖凡阶三境。

决策备注：保留既有境界名；枪毙“炼气筑基金丹”模板，因会冲掉三重宇宙体系。

条目名：命骨段

定稿内容：命骨段对应浑天星海的玄阶三境：破天境、浑天境、叩心境。破天境承受星冥重水和真空环境，开始接触真骨；浑天境熔炼高阶太古真骨，在体内凝出内星界；叩心境不是继续堆肉身，而是开始感到业火和因果反噬，是命骨体系向道业体系的过渡阀。

推演依据：设定文档第 35-38 行“命骨……真骨……肉体折磨与血腥的器官移植” + 设定文档第 54-57 行“破天境 / 浑天境 / 叩心境” + 大纲第 201 章“星相仪不仅能看天，也能解析骨文运行轨迹” + 大纲第 221 章“在体内凝出内星界雏形”

推演逻辑：破天后进入浑天星海 + 真骨移植是核心升级 + 内星界是中期关键成果 → 命骨段以破天、浑天、叩心为三境。

决策备注：枪毙“神骨血脉觉醒”，因会落入纯血统论；本体系强调移植代价与排异。

条目名：道业段

定稿内容：道业段对应宣夜无极的道阶三境：斩源境、宣夜境、无极境。斩源境先做减法，斩断旧有实质因果；宣夜境在积气中重塑自我，用学说、愿力和气运锚点避免被消融；无极境才是一念生万法、一念定因果的终极孤独。

推演依据：设定文档第 40-43 行“道业……因果总和” + 设定文档第 59-62 行“斩源境 / 宣夜境 / 无极境” + 大纲第 401 章“真正的高维晋升不是加法，而是先做减法”

推演逻辑：道业不再堆能量 + 宣夜界以积气和因果为主导 + 斩源先减法 → 道业段三境以因果锚定为核心。

决策备注：枪毙“无敌飞升境”，问题是只升级战力，不承接因果主题。

## 矿工黑话

条目名：搭骨架

定稿内容：“搭骨架”是矿工和药童对铭骨境门槛的黑话，指拓脉末期的人用劣质妖兽血肉、矿砂符粉或药渣符墨在凡骨上试刻低阶纹路。它听上去像在修命骨，实际只是盖天法界的穷人版假骨架，稍有错漏就会伤灵枢、烂骨缝。

推演依据：设定文档第 51 行“剥夺初阶妖兽血肉，尝试在凡骨上铭刻劣质符文” + 第 3 章“灵枢上的阵纹启动的时候，他也会有类似的感觉” + 大纲第 201 章“骨文门槛极高且稍有误读就会反噬肉身”

推演逻辑：凡骨铭符 + 阵纹可在灵枢上启动 + 骨文误读会反噬肉身 → 底层把未成体系的铭骨尝试叫“搭骨架”。

决策备注：枪毙“觉醒神骨”，因纯血统；“搭骨架”保留矿工口语和失败风险。

条目名：上天盖

定稿内容：“上天盖”是矿路黑话，指筑宫境末端或被天痕、星相仪提前刺激后，修士开始感到天盖压力的阶段。它不是飞升，而是人被压到壳顶，能看见壳，也更容易被壳上的阵纹反噬。

推演依据：设定文档第 52 行“筑宫境……触碰到‘叹息天盖’” + 大纲第 8 章“若天盖压死众生，那就想法子算它” + 大纲第 9 章“矿谷深处忽然裂开一道极短的天痕”

推演逻辑：筑宫触天盖 + 主角将天盖纳入账本 + 天痕能局部撕开天盖 → “上天盖”应是接触壳顶压力，不是成功破界。

决策备注：枪毙“飞升”，因第二卷才打穿天盖；黑话必须保留失败感。

## 筑基丹社会价值锚点

条目名：筑宫丹

定稿内容：为避开通用“筑基丹”模板，药鼎宗账面称它为“筑宫丹”，黑市和底层仍会把它叫“筑基丹”。一粒筑宫丹的社会价值下限锚定为：一百年清淤帮工的命价。按第 1 章日结三十文、三文一斤最次粗粮折算，若以一日一工计，一粒丹至少等于三十六万五千斤最次粗粮份额；“灵米饼”这一具体口粮名目前 10 章与设定文档未出现，故正式币值待作者定夺，系统只锁定“≥100 年帮工命价”的下限。

推演依据：第 1 章“清淤工，日结三十文” + 第 1 章“三文钱能换一斤最次的粗粮” + 设定文档第 107 行“历经百年岁月……夺得一枚至关重要的筑基丹药” + 设定文档第 52 行“筑宫境……体内筑起灵力核心”

推演逻辑：筑基丹对应筑宫突破 + 大纲明确需要百年争夺 + 底层工资和口粮已有数值 → 丹药价值下限可锚定为百年帮工命价。

决策备注：枪毙“随手一瓶丹药”；丹价必须贵到足以解释百年争夺。

## 胸口“空的地方”的生理机制

条目名：灵枢空腔

定稿内容：周衍胸口“空的地方”定稿为灵枢空腔：他的初始灵枢不是单纯差，而是杂乱灵枢中央缺一块可交换的核心腔。平时表现为胸口发紧、硬物贴胸时异常敏感、强行感气失败；星相仪残片靠血和元气启动时，正好把这处空腔当成临时接口，所以周衍能用残器看见裂点，却要付出眩晕、元气流失和更深的饥饿感。

推演依据：设定文档第 73 行“初始灵枢杂乱无章，这辈子都不可能感气成功” + 第 3 章“铜盘显然比灵枢更危险。它需要血，需要精力” + 第 3 章“拇指大小的棱角硌着胸口” + 第 5 章“草茎的凉意透过布料渗进胸口，渗得他心口发紧”

推演逻辑：灵枢杂乱无法感气 + 星相仪耗血耗精力 + 胸口反复作为硬物和灵草感受点 → 胸口空处是灵枢交换核心缺失后的残器接口。

决策备注：枪毙“天命空窍”，因像天命之子；定为病理缺口，强但有代价。

## 阿禾病弱机制

条目名：天痕压损灵枢

定稿内容：周阿禾的病不是普通寒热，也不是贵血觉醒，而是天痕压损灵枢。她的灵枢受损后无法稳定交换惰性灵气，只能靠借命草和续命药暂时补上呼吸与体温；第 9 章天痕落在矿谷附近，反推前期阿禾反复发热，是天盖局部裂纹提前压到弱灵枢的人身上。她不是天外人，只是离裂缝最近、体质最弱的本地承压者。

推演依据：第 4 章“灵枢受损……得用借命草吊着” + 第 4 章“胸口起伏得几乎看不见的呼吸” + 第 9 章“矿谷深处忽然裂开一道极短的天痕” + 设定文档第 31 行“劣等灵枢的修士强行吸纳甚至会引发肉身畸变”

推演逻辑：灵枢受损 + 需要借命草吊命 + 天痕在近处落下 + 劣等灵枢会畸变 → 阿禾病弱是天痕压力通过弱灵枢提前显影。

决策备注：枪毙“星族公主血脉”，因纯血统套路；她的特殊性来自受压位置。

## 星相仪碎片定位

条目名：残缺浑天星相仪碎片

定稿内容：星相仪碎片是高维因果与天机推演终端的残片，在盖天法界只能表现为资源扫描、裂点显影和阵纹薄弱点提示。它不是直接攻击法宝，启动方式是血、精力和灵枢空腔临时接入；碎片落在废炉区地窖，是因为药鼎宗旧炼制系统曾处理过坠星残片，失败或废弃后被丢入废炉区旧库。后续第 13、17 章的坠星货流与旧宗门弃井负责继续回收这条线。

推演依据：设定文档第 75-79 行“残缺浑天星相仪……因果与天机推演终端” + 第 3 章“废炉区……地窖……铜盘” + 第 3 章“照出矿脉裂点与阵纹薄弱处” + 大纲第 13 章“运尸和运坠星碎片的暗线” + 大纲第 17 章“坠星残片也曾从这里过手”

推演逻辑：星相仪是推演终端 + 前期只能显影裂点 + 废炉区藏铜盘 + 后续有坠星残片货流 → 地窖碎片是药鼎宗旧炼制/走私链遗留，不是随机掉落。

决策备注：枪毙“认主神器秒开”，因金手指太顺；定为残片接口，信息强、代价硬。

## 第二批自审报告

1. 本批所有条目是否都有推演链：是。每条均附前 10 章、设定文档或大纲依据。唯一未正式定值的是“灵米饼”单价，因为此前来源没有该名目；已用粗粮斤数和百年帮工命价锁定下限，避免强行编造。

2. 套路黑名单检查：没有使用“混元/太虚/九阳/无极 + 诀/经/典”功法名；没有使用血统论；“筑宫丹”回收设定文档“筑宫境”，不是通用筑基模板。

3. 五个锚点回收：本批回收周衍胸口“空的地方”的生理机制、阿禾病弱真实原因、星相仪碎片落在废炉区地窖的体系定位。矿沟死人与裴照庭玄铁动机在第三、第四批继续回收。

4. 候选内部枪毙样例：枪毙“天命空窍”，因天命之子味太重；枪毙“星族公主血脉病”，因纯血统论；枪毙“筑基丹随处流通”，因无法解释百年争夺。

5. 与前批内容矛盾检查：与第一批一致。药鼎宗、废炉区、地窖和坠星残片链条互相补强；没有把药谷改成新地名或改写季家矿路关系。
