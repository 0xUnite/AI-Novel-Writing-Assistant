## 五大锚点回收映射

条目名：周衍胸口空处

推演依据：第 3 章“拇指大小的棱角硌着胸口” + 第 3 章“铜盘显然比灵枢更危险。它需要血，需要精力” + 设定文档“初始灵枢杂乱无章”

推演逻辑：胸口反复成为残器/灵草感受点 + 灵枢杂乱 + 残器耗血耗精力 → 胸口空处是灵枢空腔和星相仪临时接口。

负责回收条目：第二批“灵枢空腔”；后续 F003、F009 持续推动。

条目名：阿禾病弱真实原因

推演依据：第 4 章“灵枢受损” + 第 9 章“天痕” + 设定文档“劣等灵枢强行吸纳会畸变”

推演逻辑：灵枢受损 + 天痕近处落下 + 弱灵枢会受压畸变 → 阿禾是天痕压损灵枢的本地承压者。

负责回收条目：第二批“天痕压损灵枢”、第三批“周阿禾角色卡”、F005。

条目名：裴照庭为什么愿意交易玄铁

推演依据：第 3 章“裴照庭说过，这东西能换几十两银子” + 第 3 章“靠闻人星阑换回星族名分” + 数据库目标“祖地、星相仪和闻人星阑三样钥匙”

推演逻辑：玄铁高价 + 裴照庭另有星族名分目的 + 三钥匙目标远高于玄铁 → 玄铁是低成本饵料和识人筹码。

负责回收条目：第三批“裴照庭角色卡”、F004。

条目名：矿沟下埋的死人

定稿内容：矿沟下埋的“死人”不是单纯被处刑的药童，而是被矿路账本抹掉的耗材：药童、矿奴和试错清淤工。尸体一部分掩埋矿沟，一部分通过尸货、骨粉灯芯和死人账进入黑市与黑矿系统，最终服务矿脉深处异常阵纹的喂养。

推演依据：第 1 章“缺半两者，罚；缺一斤者，填” + 第 5 章“灯芯要用尸坑边捡的死人骨头磨成粉” + 大纲第 6 章“这地方不只是单纯在吃人，而是在替谁养着什么” + 大纲第 16 章“借一具矿奴尸体和假账册做局”

推演逻辑：矿沟填人 + 尸骨可流入黑市日用品 + 死人账埋损耗 + 矿下喂养某物 → 矿沟死人是被资源系统二次利用的账外耗材。

负责回收条目：F001、F007。

条目名：星相仪碎片为何在废炉区地窖

推演依据：第 3 章“废炉区……地窖……铜盘” + 大纲第 13 章“运尸和运坠星碎片的暗线” + 大纲第 17 章“坠星残片也曾从这里过手” + 大纲第 31 章“星纹药鼎起血口”

推演逻辑：废炉地窖藏铜盘 + 后续有坠星残片暗线 + 药鼎旧阵上抬 → 碎片是药鼎宗旧炼制/坠星货流失败后遗留的残片。

负责回收条目：第一批“药谷与废炉区”、第二批“残缺浑天星相仪碎片”、F003。

## 11-30 章 beat 规划

11-15 章目标：让周衍进入黑市深水区，先被抽成、差价、禁货箱和丹方缺味打脸，证明“算到”不等于“拿到”。

16-20 章目标：用尸体、税契、旧井和暗路路引把宁见微从旁观者逼成灰盟友，同时让坠星残片货流露出第一截。

21-25 章目标：把压力从药谷和黑市推进周家门口，让税刀、祖宅旧物、顾家名字和阿禾被带走，把私人药钱线升级成家门生死线。

26-30 章目标：周衍借族老贪念、父亲旧册、死人账和祠堂封井反击，守住周家并接回阿禾，章末用宗门内门调令把地方胜利清零。

## 第四批自审报告

1. 本批所有条目是否都有推演链：是。伏笔清单和 11-30 规划均来自前 10 章、大纲 11-30、设定文档或数据库角色表；没有新增无依据势力。

2. 套路黑名单检查：没有新增宗门套路名；没有新增功法名；没有把周衍写成废柴退婚或家族弃子，周家线定位为税刀、旧账和阶级压迫。

3. 五个锚点是否都被回收：全部回收。胸口空处由第二批与 F003/F009 回收；阿禾病因由第二、第三批与 F005 回收；裴照庭玄铁动机由第三批与 F004 回收；矿沟死人由 F001/F007 回收；星相仪地窖由第一、第二批与 F003 回收。

4. 候选内部枪毙样例：枪毙“矿沟死人是妖兽吃剩”，因为大纲明确死人账和喂养某物，不是野怪；枪毙“税吏只是路人”，因为第 21-25 章承担周家门口压力；枪毙“屠麻子是长期伙伴”，因为大纲功能是黑市猎手压迫。

5. 与前批内容矛盾检查：与前三批一致。药鼎宗旧炼制链、季寒庐死人账、阿禾天痕病因、裴照庭星海三钥匙目标均被伏笔清单继续承接。
